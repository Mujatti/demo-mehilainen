# AddSearch AI Search Experience Starter App

## Mehiläinen demo variants

- Search-first branded demo: `/?config=mehilainen-brand`
- Care navigator variant with comparison: `/care?config=mehilainen-care`
- Chat-first variant: `/chat?config=mehilainen-brand`

## URL overrides

These work on all variants, including `/care` and `/chat`:

- `?site_key=YOUR_SITE_KEY`
- `?siteKey=YOUR_SITE_KEY`
- `?config=mehilainen-brand`
- `?config=mehilainen-care`

Query parameters are preserved when moving between variants from the built-in navigation.
